# 国立青少年教育施設 環境マッピングマップ

国立青少年教育施設（全29施設）の環境データ（温度・湿度・照度・CO2）をインタラクティブに可視化するWebアプリケーションです。高校「情報II」データサイエンス授業での活用を想定しています。

## 構成

```

```

## 主な機能

- **地図表示**: 全国29施設の位置と快適スコアをリアルタイム表示（Leaflet.js）
- **季節・時間帯切替**: 春夏秋冬 × 朝昼夕夜の16パターンのデータ閲覧
- **データサイエンス分析**:
  - 📊 分布（ヒストグラム・箱ひげ図・基本統計量）
  - 📈 相関（散布図・回帰直線・相関係数）
  - 📉 時系列（施設別の時間帯推移）
  - ⚠️ 異常値検知（平均±2σ）
  - 💾 CSVエクスポート
- **シリアル接続**: Schoomy センサーからのリアルタイムデータ取得（Web Serial API）

## 授業での活用

1. 季節・時間帯を変更して、温度や照度の変化パターンを観察
2. 相関タブで「緯度×温度」を選び、地理的な傾向を分析
3. 分布タブで統計量（平均・中央値・標準偏差）を確認
4. CSVでデータをダウンロードし、スプレッドシートで独自分析
5. Schoomyセンサーで教室の環境データをリアルタイム計測

## 技術情報

- HTML/CSS/JavaScript のみ（サーバー不要）
- Leaflet.js（地図ライブラリ）
- Web Serial API（センサー接続）
- Canvas API（グラフ描画）
